# ⚠️ Important: Installing Node.js

To run the frontend, you need to install Node.js.

## Download Node.js

1. Go to: https://nodejs.org/
2. Download **LTS version** (version 18 or higher recommended)
3. Run the installer
4. Follow the installation instructions
5. Restart PowerShell/VS Code

## Verify Installation

After installing Node.js, open a new PowerShell terminal and run:

```powershell
node --version
npm --version
```

You should see the versions of Node.js and npm.

## After Installing Node.js

Return to [README.md](./README.md) and follow the instructions for installing frontend dependencies.
